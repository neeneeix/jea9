			<font style="font-size:30px;"><i>Formatting Text & Layout with CSS</i></font><br>
			<font color="#464f62">Typography and Box Model</font>
			<p style="padding:5px;">Selectors, div, span, and link are foundational CSS tools that allow you to target specific HTML elements, structure web content, and define interactive styles for enhanced design and functionality.</p><img src="GIF7.gif"><br><br>
<!-- INFO STUFF -->
<font color="#5f6677"  size="3.5"style="text-align:left;">Typography Basics</font><p>CSS typography controls how text is styled and displayed on a webpage using properties like font family, size, weight, and style.</p>

<br><font color="#999fad" size="2" style="text-align:left;">Key Typography Properties:</font>
<ul type="disc" style="text-align:left;">
<li><u>font-family: </u>Sets the typeface (e.g., Arial, Courier).</li><br>
<li><u>font-style: </u>Specifies normal, italic, or oblique text.</li><br>
<li><u>font-weight: </u>Adjusts text thickness (e.g., normal, bold, 100â€“900).</li><br>
<li><u>font-size: </u>Controls text size (e.g., 16px, medium, 150%).</li><br>
<li><u>font-variant:</u> Enables small caps (e.g., small-caps).</li></ul><br><br>

<br><font color="#999fad" size="2" style="text-align:left;">Text Properties:</font>
<ul type="disc" style="text-align:left;">
<li><u>text-align: </u>Aligns text (e.g., left, center, justify).</li><br>
<li><u>text-decoration: </u>Adds styles like underline, line-through, or none.</li><br>
<li><u>text-transform: </u>Changes text case (e.g., uppercase, lowercase).</li><br>
<li><u>line-height: </u>Adjusts spacing between lines.</li><br>
<li><u>text-indent: </u>Indents the first line of a paragraph.</li></ul><br><img src="GIF5.gif"><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">List Styling</font><p>CSS provides control over bullet styles, positioning, and custom images for lists.</p><br>
<br><font color="#999fad" size="2" style="text-align:left;">Key List Properties:</font>
<ul type="disc" style="text-align:left;">
<li><u>list-style-type:</u> Defines bullet type (e.g., disc, circle, decimal).</li><br>
<li><u>list-style-image:</u> Replaces bullets with a custom image.</li><br>
<li><u>list-style-position</u>: Positions bullets (e.g., inside, outside).</li></ul><br><img src="GIF3.gif"><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Pseudo-Elements</font><p>Pseudo-elements style specific parts of an element, such as the first letter or line.</p><br>
<br><font color="#999fad" size="2" style="text-align:left;">Common Pseudo-Elements:</font>
<ul type="disc" style="text-align:left;">
<li><u>:first-letter:</u> Styles the first letter.</li><br>
<li><u>:first-line:</u> Styles the first line.</li><br>
<li><u>:before / :after:</u> Inserts content before or after an element.</li></ul><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Box Model</font><p>The CSS box model includes margins, borders, padding, and content, defining how elements are spaced and aligned.</p><br>
<br><font color="#999fad" size="2" style="text-align:left;">Key Properties:</font>
<ul type="disc" style="text-align:left;">
<li><u>margin:</u> Space outside the element.</li><br>
<li><u>padding:</u> Space inside the element, between content and the border.</li><br>
<li><u>border:</u> Style and width of the element's edges.</li></ul><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Positioning and Layout</font><p>CSS positioning and layout properties determine how elements are placed on a webpage.</p><br>
<br><font color="#999fad" size="2" style="text-align:left;">Position Values:</font>
<ul type="disc" style="text-align:left;">
<li><u>static:</u> Default position.</li><br>
<li><u>relative:</u> Positioned relative to its normal position.</li><br>
<li><u>absolute:</u> Positioned relative to the nearest positioned ancestor.</li></ul><br>
<li><u>fixed:</u> Stays in place during scrolling.</li><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Offset Properties:</font><p>Control distances from top, right, bottom, and left.</p><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Float:</font><p>Positions an element to the left or right, with surrounding text wrapping around it.</p><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Background Styling</font><p>CSS allows full customization of element backgrounds with color, images, and positioning.</p><br>
<br><font color="#999fad" size="2" style="text-align:left;">Key Background Properties:</font>
<ul type="disc" style="text-align:left;">
<li><u>background-color:</u> Sets the background color.</li><br>
<li><u>background-image: </u>Adds a background image.</li><br>
<li><u>background-repeat:</u> Controls tiling (e.g., repeat, no-repeat).</li><br>
<li><u>background-position:</u> Specifies image placement (e.g., top left).</li></ul><br>

<font color="#5f6677"  size="3.5"style="text-align:left;">Borders</font><p>Borders outline elements with various styles and thicknesses.</p><br>
<br><font color="#999fad" size="2" style="text-align:left;">Key Properties:</font>
<ul type="disc" style="text-align:left;">
<li><u>border-width: Thickness</u>  (e.g., thin, 10px).</li><br>
<li><u>border-style: Style </u> (e.g., solid, dashed, double).</li><br>
<li><u>border-color:</u>  Specifies border color.</li></ul><br>
