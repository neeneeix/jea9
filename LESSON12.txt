			<font style="font-size:30px;"><i>Intro to CSS</i></font><br>
			<font color="#464f62">Cascading Style Sheets</font>
			<p style="padding:5px;"><u>Cascading Style Sheets (CSS)</u> is used to style HTML elements on a webpage, defining their layout, design, and visual effects.<br><ul type="disc"><li>HTML (Content Layer) provides the structure and content of the page</li><li>CSS (Presentation Layer) to make websites visually appealing and functional</li><li>JavaScript (Behavior Layer) for overall interactivity and dynamic content</li></ul>CSS makes websites visually appealing, easier to maintain, and more functional across various devices.</p><br><img src="GIF9.gif"><br><br>

<font color="#5f6677" style="text-align:left;">How is CSS applied?</font><ol align="left"><li>Inline CSS: Style directly within HTML tags using the style attribute</li><li>Style defined within the style tag in the head of the HTML document.</li><li>Style defined in a separate .css file and linked to the HTML file using the link tag.</li></ol>

<font color="#5f6677" style="text-align:left;">Benefits of CSS:</font><br><p align="left" style="padding-left:15px; padding-right:15px;">
<input type="checkbox" id="line1" name="point1" value="1">
<label for="point1">Saves time and effort in maintaining a consistent design across pages.</label><br>
<input type="checkbox" id="line2" name="point2" value="2">
<label for="point2"> Allows the layout to be more flexible, including precise control over positioning and layering of elements.</label><br>
<input type="checkbox" id="line3" name="point3" value="3">
<label for="point3"> Makes it easy to update the look and feel of a website by editing a single file.</label><br></p><br>

<font color="#5f6677" style="text-align:left;">Creating and Linking CSS:</font><br><p>To create an external CSS file: Write CSS in a text editor, save it with a .css extension, and link it to the HTML file using the "link" tag</p><br>

<font color="#5f6677" style="text-align:left;">CSS Syntax:</font><br><p>A CSS rule consists of a selector (which element to style) and declarations (how to style it, e.g., font-family, color)</p><br>

<font color="#5f6677" style="text-align:left;">Grouping Selectors:</font><br><p>Multiple selectors can be grouped in one rule to apply the same style to several elements, separated by commas (e.g., h1, p {color: blue;}).
