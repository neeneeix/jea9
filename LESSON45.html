Selectors, Div, Span, and Link
Properties and Dimensions

Selectors, div, span, and link are foundational CSS tools that allow you to target specific HTML elements, structure web content, and define interactive styles for enhanced design and functionality.



Display Property

The display property determines how elements are arranged on the page:

  • Block-Level Elements: Display vertically, with each element starting on a new line
  • Inline-Level Elements: Display horizontally, flowing with surrounding content

Common Display Values


Div and Span Tags
  1. Div Tag
    • A block-level element used to group and style sections of a webpage
    • Adds spacing (line breaks) before and after its content
  2. Span Tag
    • An inline-level element used to style or emphasize parts of text

  • 'div' is best for large sections or layout structures
  • 'span" best for small inline styles or text snippets


Selectors: Classes and IDs
  1. Class Selector:
    • Used to apply styles to multiple elements.
    • Defined with a period (.)
  2. ID Selector:
    • Targets a unique element on a page.
    • Defined with a hash (#).

Grouping, Nesting, and Combining Selectors
  1. Grouping Selectors:
    • Apply the same style to multiple elements by separating selectors with commas.
  2. Nesting Selectors:
    • Style elements inside another element.

Pseudo-Classes and Links

Pseudo-classes define styles for specific states of elements. They are defined with a colon (:)


Pseudo-Classes for Links:
  • :link- Styles unvisited links

  • :visited- Styles links after they've been visited

  • :hover- Styles elements when hovered over by a cursor

  • :active- Styles elements when they are clicked

  • :focused- Styles elements when focused (e.g., an input field)
